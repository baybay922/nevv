## 代码规范
- 常量使用全大写字母，并用下划线分割 GLOBAL_NAME="name"
- 变量和函数小驼峰方式，变量名词开头，函数动词开头
- 构造函数class大驼峰方式
- 一行代码不超过80个字符，编辑器会换行
- 字符串，统一使用单引号，多行或需要携带变量使用反引号,props属性传参使用双引号
- null当做对象占位符
- 避免使用undefined
- 创建对象时使用：{}
- 创建数组时使用：[]
